MEDIO AMBIENTE - UNA ACCION VALE MAS QUE MIL PALABRAS

Para contribuir al cuidado del medio ambiente podemos apoyar con una serie de pasos:

Paso 01: Tomar conciencia.
Paso 02: Reducir el consumo de recursos plasticos.
Paso 03: Usar adecuadamente el agua, para no desperdiciarla.
Paso 04: Usar el transporte publico u otra alternativa de transporte.
Paso 05: Reducir el uso de energias electricas.
